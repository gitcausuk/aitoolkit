# 1.2.0

- Removed pseudo-precise numerical VIVA readiness scores.
- Added transparent qualitative review indicators and explanations.
- Added “Why this result?” for permission suggestions.
- Clarified that disciplinary matching, exposure labels, route ordering and marking weights are deterministic heuristics or editable defaults.
- Added visible methodology guidance while preserving the existing design.

# Changelog

## 1.1.0 - 20 July 2026

- Editorial pass to reduce word count: shorter status strip, hero lead, footer and source note; caveats consolidated rather than repeated.

- Renamed the application from Oxford Assessment Designer to AI Designer.
- Removed language implying an official or complete Oxford assessment taxonomy; the 40-format list is now described as a reference of commonly used formats.
- Added consultation framing: the tool is a working prototype open to departmental input, with room for departmental format lists and competence frameworks to be incorporated as these are agreed.

## 1.0.0 - 20 July 2026

- Initial cross-disciplinary assessment redesign application.
- Added browser-only PDF, DOCX and text extraction.
- Added the 40-format reference data.
- Added VIVA diagnostics and four Toolkit permission levels.
- Added three redesign routes and copy-ready assessment brief generation.
- Added Word, Markdown, JSON and print export.
